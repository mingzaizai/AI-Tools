## 🔧 开发注意事项

### Canvas 绘制问题
- 相框样式需要在 Canvas 中正确绘制
- 阴影效果在某些情况下可能不显示
- 下载功能需要使用 Blob URL 而非直接跳转

### 日期解析
- 不支持的日期格式需要友好提示
- 同时输出秒级和毫秒级时间戳

### 样式兼容性
- 复杂的 CSS 效果（如 boxShadow）在 Canvas 中需要特殊处理
- 不推荐的样式及时移除，避免用户困惑
